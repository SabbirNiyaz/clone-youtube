# YouTube Clone

## Project Overview
This project is a web-based **YouTube Clone** built using HTML, CSS, JavaScript, and Tailwind CSS. It uses a demo API to fetch video data and allows users to browse, search, and view videos in a layout similar to YouTube.

## Features
- Search for videos using a demo API
- Display trending and popular videos
- Video detail view with thumbnail and description
- Responsive layout using Tailwind CSS
- Dynamic content loading with JavaScript

## Project Structure

/clone-youtube
│
├─ index.html # Main landing page
├─ style.css # Custom styles
├─ tailwind.config.js # Tailwind configuration
├─ script/test.js # Demo API and dynamic content logic
└─ assets/ # Images, icons, and media


## Technology Stack
- HTML / CSS - Markup and styling
- Tailwind CSS - Utility-first CSS framework
- JavaScript - Client-side scripting
- Demo API - Provides sample video data
